# FSLogix Management Scripts

A collection of PowerShell scripts and tools for managing FSLogix profile containers and ODFC (Office Container) deployments in Azure and hybrid environments.

## Available Scripts

### [MigrateContainers](./MigrateContainers/)

Migrate FSLogix profile containers between Azure Storage Accounts with VHD/VHDX conversion, ACL management, and optional folder renaming.

[View Full Documentation →](./MigrateContainers/README.md)

## Getting Started

```powershell
git clone https://github.com/shawntmeyer/FSLogix.git
cd FSLogix
```

Navigate to the specific script folder and review the README for detailed usage instructions.
